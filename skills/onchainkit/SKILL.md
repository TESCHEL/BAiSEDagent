# OnchainKit Skill

## Description
OnchainKit is a React component library and TypeScript utilities for building onchain applications.

## Capabilities
- Wallet connection and management
- Transaction handling
- Identity resolution (ENS, Basename)
- Token operations
- NFT interactions

## Usage
This skill enables Clawdbot to assist with OnchainKit-related development tasks.
